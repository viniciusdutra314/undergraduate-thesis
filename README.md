# Undergraduate Thesis: Graph Traffic Simulation

This repository contains the code, data analysis, and manuscript for my undergraduate thesis, which focuses on simulating traffic over complex networks.

The project uses Rust (high performance simulation), Julia (data analysis), and Typst (final document)
## Project Architecture 

1. **Rust (`GraphTraffic-rs/`) - The Simulation Engine**  
   This directory is a Git submodule containing a high-performance simulation engine written in Rust. It compiles into a CLI tool (`graph_traffic`) that accepts simulation configurations via JSON and efficiently saves the detailed simulation output to HDF5 files. https://github.com/viniciusdutra314/GraphTraffic-rs

2. **Julia (`src/`) - Orchestration & Data Analysis**  
   The Julia code acts as the bridge between the simulations and the final thesis. 
   - **Configuration:** It defines the graph topologies, sets up simulation configurations, and spawns the Rust engine (see `src/GraphTraffic.jl`).
   - **Analysis:** It reads the HDF5 results generated by Rust and generates plots using `Makie` 
3. **Typst (`thesis/`) - Document Typesetting**  
   The actual thesis manuscript (`main.typ`) and presentation slides (`slides.typ`) are written in Typst. The Typst code references the SVG plots and CSV tables produced by Julia, automatically keeping the document up to date with the latest simulation results
